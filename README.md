# 🐍 贪吃蛇学习项目

欢迎来到贪吃蛇学习项目！这是一个适合初中生学习编程的有趣项目。

## 📖 项目介绍

这是一个使用 HTML、CSS 和 JavaScript 制作的经典贪吃蛇游戏。
- ✅ **无需安装**：双击 `snake-game.html` 就能玩
- ✅ **详细注释**：每段代码都有中文说明
- ✅ **Apple 风格**：简洁优雅的设计

## 🎮 如何开始

1. 双击打开 `snake-game.html` 文件
2. 按**空格键**开始游戏
3. 使用**方向键 ↑ ↓ ← →** 控制蛇的移动

## 🎯 游戏规则

- 🐍 控制小蛇吃掉红色的食物
- 📈 每吃一个食物，蛇会变长，分数增加
- ⚡ 分数越高，蛇移动越快
- ❌ 撞到墙壁或自己身体就游戏结束
- ⏸️ 按**空格键**可以暂停/继续

## 📊 功能特性

- ✨ 流畅的游戏动画
- 🎯 实时计分系统
- 🏆 最高分记录（自动保存在浏览器中）
- ⚡ 难度递增（速度会越来越快）
- 🎨 简洁的 Apple 风格设计

## 📚 学习要点

打开 `snake-game.html` 你可以学到：
1. **HTML**：网页结构
2. **CSS**：样式和动画设计
3. **JavaScript**：游戏逻辑编程
   - 变量和数据类型
   - 函数的使用
   - 条件判断（if-else）
   - 循环（for 循环）
   - 键盘事件监听
   - Canvas 画布绘制

## 🚀 下一步优化方向

学会基础后，你可以尝试：
- 🎵 添加背景音乐和音效
- 🌈 更换不同的颜色主题
- 🎁 添加特殊道具（加速、减速、穿墙等）
- 📱 适配手机触摸操作
- 👥 双人对战模式

## 💡 遇到问题？

- 确保使用现代浏览器（Chrome、Edge、Safari 等）
- 打开浏览器的开发者工具（按 F12）查看错误信息
- 仔细阅读代码中的注释
