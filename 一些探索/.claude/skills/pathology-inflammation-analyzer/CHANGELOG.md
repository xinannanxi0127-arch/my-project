## 版本命名规范

本项目遵循 [语义化版本](https://semver.org/zh-CN/) 规范：

- **MAJOR** (主版本号): 不兼容的 API 变更
- **MINOR** (次版本号): 向后兼容的功能性新增
- **PATCH** (修订号): 向后兼容的问题修正
